# fintrack-cli-finance-manager
FinTrack is a command-line personal finance manager built with Python, SQLite, and SQLAlchemy ORM. It allows users to manage expenses, categories, budgets, subscriptions, and generate spending reports using both ORM operations and raw SQL queries.

FinTrack is a CLI-based personal finance management system developed using Python and SQLAlchemy ORM. The application enables users to record expenses, organize categories, track subscriptions, manage budgets, and generate analytics reports. This project demonstrates practical database design, CRUD operations, ORM relationships, and raw SQL reporting.
